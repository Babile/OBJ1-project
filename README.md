# OBJ1-project

Autor: Nemanja Babic 
Indeks: NRT-120/18

Kontrole u igri:
1. Tab  - sacuva igru u text fajl
2. Alt - loduje igru koju smo sacuvali
3. Shift - sortira objekte
4. ESC - za izlazak iz igre

Koristeci levi klik mis mozemo da selektujemo objekte ako postoje i desnim klikom pomerati na sledecu destinaciju. Ukoliko nije ni jedan objekat selektovan, desnim klikom pravimo novi objekat.
